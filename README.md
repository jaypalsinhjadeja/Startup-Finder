# Startup Finder AI

AI-powered startup opportunity discovery and research tool.

* Discover startup opportunities from keywords
* Research market demand and trends
* Analyze startup potential
* Generate multiple startup ideas
* Opportunity scoring system
* AI-assisted startup research workflow

## Tech Stack

* HTML
* CSS
* JavaScript

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start researching startup opportunities
